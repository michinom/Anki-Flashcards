# Anki Deck for KCCUK Sejong 2

This Anki deck contains study notes from **Sejong 2 (세종한국어 2)**.

## Card Format

- The **front** of each card shows the **native language**.
- The **back** of each card shows the **target language**.
- This setup is designed to help with **active recall**.
- The front/back order **can be swapped in the Anki settings** if you prefer the opposite direction.

## Display & Audio

- This deck is designed to work well in both **dark mode and light mode**.
- All audio was produced using **ElevenLabs.io AI text-to-speech** with **Jin Geon Song's voice**.
- It is **recommended to disable audio autoplay** in the Anki settings for better control during study.

## How to Import

1. Download the `.apkg` file to your device.
2. Open **Anki**.
3. Click **File → Import**.
4. Select the downloaded `.apkg` file.
5. Confirm the import settings and sync your collection if needed.
